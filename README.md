# ArvyaX AI-Assisted Journal System

### How to Run
1. **Backend:**
   - cd backend
   - pip install -r requirements.txt
   - uvicorn main:app --reload
2. **Frontend:**
   - cd frontend
   - npm install
   - npm start

### Tech Stack
- **Backend:** Python (FastAPI)
- **Frontend:** React.js
- **Database:** SQLite (Local)
- **AI:** Integrated via API (Gemini)
